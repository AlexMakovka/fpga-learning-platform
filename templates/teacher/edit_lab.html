Паспорт лабораторной работы

Для Verilog-заданий этот код автоматически подставляется студенту во встроенный HDL-редактор на странице лабораторной работы.
Открытые тесты показываются студенту подробно и используются для учебной диагностики ошибок.
Скрытые тесты используются для итоговой оценки. Студент по умолчанию видит только количество пройденных скрытых тестов.
Обычно эту настройку лучше не включать, чтобы скрытые проверки оставались закрытыми.
Если студент уже использовал больше попыток, старые попытки сохранятся, но новые будут ограничены этим числом.
